## About

The **Employee Management System** is a web-based application built using **React** and **Bootstrap** for frontend UI.  
It allows **Admin, Manager, and Employee roles** to manage employee and department data efficiently.  
Features include adding, editing, viewing employees and departments, role-based access, and profile management.

---

## Features

- **Role-Based Login:** Separate dashboards for Admin, Manager, and Employee.
- **Employee Management:** Create, view, edit, and delete employee records.
- **Department Management:** CRUD operations for departments.
- **Employee Profiles:** View detailed employee information.
- **Responsive UI:** Bootstrap-based styling for desktop and mobile.
- **Routing:** React Router for seamless navigation between pages.
- **Manager Dashboard:** Special dashboard and views for managers.
- **Employee Dashboard:** Simplified dashboard and profile access for employees.

---

## Roles Overview

| Role    | Description            | Permissions                                           |
|---------|------------------------|------------------------------------------------------|
| Admin   | System manager         | Manage employees, departments, and view all profiles|
| Manager | Department manager     | Manage employees in their department                |
| Employee| Staff                  | View and update personal profile                     |
